## Integracion de React en Laravel
- Una vez creado el proyecto con *Laravel*, dentro de la carpeta del proyecto instalar:
~~~
composer require ui
~~~
- Luego instalar react:
~~~
php artisan ui react
~~~
- Correr los comandos
~~~
npm install
~~~
